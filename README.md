# MP-Backend

## Executando o projeto

1 - Primeiro instale as dependencias, para isso execute o comando abaixo:

```sh
    pip install -r requirements.txt
```

2 - Execute o servidor. Para isso execute o comando abaixo:
```sh
    make start
```
